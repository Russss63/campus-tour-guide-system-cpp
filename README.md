# 校园导游系统 (Campus Tour Guide System)

## 项目简介
本项目为基于 C++ 实现的校园导游系统，通过图结构建模校园景点之间的连接关系，并支持查询任意两个景点之间的最短路径。

项目主要用于练习数据结构、图论算法以及 C++ 程序设计能力。

## 主要功能
- 校园景点信息展示
- 景点信息查询
- 景点信息更新
- 任意两景点之间最短路径查询
- 控制台交互式菜单

## 技术实现
- 编程语言：C++
- 数据结构：邻接矩阵（Adjacency Matrix）
- 图算法：Floyd 最短路径算法
- 程序结构：模块化函数设计

## 项目背景
该项目为课程设计项目，通过构建校园景点图模型，实现校园景点信息管理与最短路径查询功能，旨在加深对图结构与最短路径算法的理解。
